### :pushpin: 소개
[가지가지]는 자체 선불충전금 서비스인 '가지머니'를 통해 안정성을 고려한 중고거래 서비스를 제공합니다.

### :pushpin: 기능 요약
- 웹사이트 이용자는 거래 약속과 결제를 채팅창 안에서 한 번에 진행할 수 있습니다.
- 채팅방에서의 선결제 시스템으로 직거래 노쇼, 네고를 방지할 수 있습니다.
- '가지머니'를 통해 계좌번호 노출 없이 안전한 중고거래가 가능합니다.

### :alarm_clock: 개발 기간
2023년 5월 1일 ~ 2023년 6월 9일(6주)
